Tutoriel d’Inscription aux Cours d’Escalade

Ce guide vous explique, étape par étape, comment vous inscrire aux cours d’escalade du CAF La Roche - Bonneville.

Avant de commencer :

  • Assurez-vous de respecter les dates d’ouverture des inscriptions.
  • Préparez votre moyen de paiement (carte bancaire via HelloAsso).
  • La licence CAF est obligatoire (à prendre après validation de votre inscription).

📌 Processus d’Inscription en un coup d'œil

  1. L’utilisateur remplit son formulaire sur le site.
  2. Un lien de paiement est envoyé via HelloAsso.
  3. L’utilisateur effectue son paiement.
  4. Confirmation du paiement envoyée au site.
  5. Le club attribue les cours (±1 semaine de délai).
  6. Inscription validée dans la messagerie interne.
  7. L’utilisateur peut vérifier son inscription.
  8. E-mail d’information envoyé fin août.

1️⃣ Accéder à la plateforme d'inscription

Cas 1 : Vous êtes déjà licencié au CAF La Roche Bonneville

✅ Connectez-vous avec votre numéro de licence ici :

caflarochebonneville.fr

Cas 2 : Vous n’êtes pas encore licencié

  • ✅ Attendez la validation de votre inscription avant de prendre votre licence (après le 1er septembre).
  • ✅ Connectez-vous via votre adresse e-mail :

Espace d’inscription → Connexion

❗ Si pas d’e-mail, vérifiez les indésirables (spam).

2️⃣ Coordonnées

Une fois connecté, cliquez sur „Étape suivante” ou „Coordonnées” :

Remplissez vos coordonnées (utilisées exclusivement pour le suivi). :

Vous étiez déjà inscrit l’année dernière

✅ Vos coordonnées sont déjà enregistrées par la FFCAM. Pour les modifier, connectez-vous sur le site de la fédération.

3️⃣ Sélectionner la personne à inscrire

Cliquez sur „Étape suivante” ou „Inscription” :

Cas 1 : Vous étiez déjà inscrit l’année dernière

✅ Sélectionnez votre nom dans la liste.

Cas 2 : Première inscription

✅ Entrez les informations personnelles (nom, prénom, date de naissance).

📌 La date de naissance détermine le niveau de cours.

📝 Processus d’attribution des créneaux

L’attribution des créneaux est un processus clé. Pour maximiser vos chances, choisissez bien vos créneaux et votre préférence de moniteur.

  1. Inscription validée après réception du paiement.
  2. Classement selon vos préférences.
  3. Test du 1er choix : si disponible ✅, sinon test suivant.
  4. Si aucun disponible → liste d’attente ⏳.

4️⃣ Choisir un cours

🧒 Débutants / Confirmés (jeunes)

  • Tous les nouveaux commencent Débutant (assurage, fondamentaux, sécurité).
  • Après 1 an, passage en Confirmé.

👨‍🎓 Loisirs / Perfectionnement (adultes)

Loisir : accessible à tous, rythme libre.
Perfectionnement : base technique souhaitée, travail spécifique.

Groupes soumis à l’approbation

  • Perf.
  • Pré‑compétition 6‑8 ans
  • Compétition U12‑U14
  • Compétition U16‑Vétéran

6️⃣ Indiquer ses préférences créneaux et moniteurs

🕝 Créneaux horaires

  • Cliquez sur ⬆ pour monter, ⬇ pour descendre, ✖ pour supprimer.
  • Attention : limiter vos choix réduit vos chances.

🙍‍♂️ Moniteurs

  • Oui : priorité à l’ancien moniteur.
  • Non : autre moniteur selon dispo.
  • Peu importe : flexibilité maximale.

Si un créneau est (Complet), il n’y a plus de places.

Un créneau marqué disponible peut déjà être en cours de remplissage.

7️⃣ Valider les informations et accepter le règlement

  1. Droit à l’image : autorisez ou refusez.
  2. Vérifiez vos infos et créneau choisi.

Cliquez sur “Valider cette inscription”.

⚠️ Une inscription n’est validée qu’après paiement !

8️⃣ Paiement de l'inscription

Le paiement s’effectue en ligne via HelloAsso.

✅ Une fois le paiement effectué, votre inscription est finalisée.

Délai d’attribution : environ 1 semaine (notification interne).
